Rules are the foundation

Selectors & Declarations

If you scroll to the bottom of the css above, you should see these two rules at the end:

img { max-width: 100% }
img.small { width: 100px }

'img' refers to image elements, and 'img.small' refers to image elements of the class 'small'. Everything within { } are declarations that get applied to select elements.
